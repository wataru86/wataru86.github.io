Hakyllを始める

Posted on May 2, 2018

はじまる

$ git clone https://github.com/jaspervdj/hakyll.git
$ cd hakyll
$ stack install
$ hakyll-init my-site
$ cd my-site
$ stack init
$ stack build
$ stack exec site build
$ stack exec site watch

ってするとhttp://localhost:8000/でプレビューできるかもしれない．
